\section{Per-Process Model Performance}
\label{sec:per-process}

The GAN was trained independently on $10^5$ Monte Carlo simulations of the 2-inverter circuit for each of the nine PTM process models. All runs used the same architecture, learning-rate schedule, and 100-epoch budget. Final metrics are summarized in Table~\ref{tab:per-process-metrics}. In every run the training loss and average test loss are within $\sim$1\% of each other, indicating the model is not overfitting --- the cross-process variation is driven by representation and target-distribution effects rather than generalization gap.
